## How to Use These Files

### For Developers (Rebuilding Forms)

1. **Read the Markdown file** for human-readable documentation
2. **Use the JSON file** for exact field specifications
3. **Follow "Rebuild Instructions"** section in each markdown file

### For LLMs (Automated Processing)

The JSON files contain structured data suitable for:
- Generating form HTML
- Creating validation logic
- Building similar forms
- Understanding form relationships

### JSON Structure Convention

Each JSON file follows this structure:
```json
{
  "_metadata": {

```

## Rebuilding the Website Forms

### Prerequisites
- WordPress installation
- Required plugins:
  - WooCommerce
  - Contact Form 7
  - WooCommerce Square Payment Gateway
- External accounts:
  - Crisp account
  - Mailchimp account
  - Square account

### Installation Order
1. Install WordPress with theme
2. Install and configure WooCommerce
3. Set up Square payment gateway
4. Install Contact Form 7
5. Create contact form using specifications
6. Add Crisp chat widget code
7. Add Mailchimp newsletter form
8. Implement React course search (custom development)
