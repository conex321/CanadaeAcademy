# Course Coding System

The Ontario secondary education system is built on the concept of students earning credits in both compulsory and elective areas. All courses offered through Canada e Academy use a 5-character coding system established and recognised by the Ontario Ministry of Education.

---

## How to Read a Course Code

Every Ontario course code follows this pattern:

**[Subject Area — 3 characters][Grade Level — 1 digit][Course Type — 1 letter]**

### Example: MCV4U

| Position | Character | Meaning |
|----------|-----------|---------|
| 1st | **M** | Major area of study — Mathematics |
| 2nd–3rd | **CV** | Course descriptor — Calculus and Vectors |
| 4th | **4** | Grade level — Grade 12 |
| 5th | **U** | Course type — University Preparation |

---

## Grade Level (4th Character)

| Digit | Grade |
|-------|-------|
| 1 | Grade 9 |
| 2 | Grade 10 |
| 3 | Grade 11 |
| 4 | Grade 12 |

---

## Course Type (5th Character)

### Grades 9 and 10

| Letter | Type | Description |
|--------|------|-------------|
| D | Academic | Emphasises theory and abstract thinking |
| P | Applied | Emphasises practical, hands-on applications |
| W | De-streamed | Ontario's new de-streamed courses (Grade 9) |
| O | Open | Appropriate for all students |

### Grades 11 and 12

| Letter | Type | Description |
|--------|------|-------------|
| U | University Preparation | Prepares students for university programmes |
| C | College Preparation | Prepares students for college programmes |
| M | University/College (Mixed) | Prepares for both university and college |
| O | Open | Appropriate for all students |
| E | Workplace Preparation | Prepares for direct entry to the workplace |
